# random-word-uuid

> Return random words.

## Install

```bash
$ npm install random-word-uuid
```

## Usage

```javascript
const randomWordUUID = require('random-word-uuid');

console.log(randomWordUUID()); // zonk
```

Define length of random words.

```javascript
const randomWordUUID = require('random-word-uuid');

console.log(randomWordUUID(3)); // hay
console.log(randomWordUUID(4)); // yams
```
